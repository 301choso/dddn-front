# dddn-front

반려동물 사진공유 홈페이지(front)

## 사용스택
- Vue 3
- bootstrap
- axios
- element-plus


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
